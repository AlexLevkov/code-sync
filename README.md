## CodeSync 
CodeSync is a real-time code-editing app for mentor-student interaction
<br/>
[Run app 🚀](https://codesync.alexlevkov.com/)

### 🖥️ Main Features:
* Real-time code editing
* Chat interface for efficient communication
* User list to track active participants in real-time
* Hint system for quick and easy problem-solving
* Animations features to enhance user experience
* Easy code block management

### ⚙Tech Stack :

#### Frontend:
* React
* React Router 
* React Query 
* Bootstrap
* Animate.css
* Sass
* Socket.io 

#### Backend:
* Node.js 
* MongoDB
* Express 
* Cors 
* Socket.io
